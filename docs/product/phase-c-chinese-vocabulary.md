# 中文产品词汇

## 目标

冻结客户可见的中文词汇，让 UI、测试、文档在同一套产品语言里表达。

## 一级导航

- `收件箱`
- `任务`
- `日历`
- `频道`
- `AI 同事`
- `设置`

## 工作区核心概念

- `工作区`
- `工作台`
- `频道时间线`
- `协作`
- `置顶`
- `审批`
- `模型连接`
- `能力`
- `记忆`

## 任务状态

- `待办`
- `进行中`
- `待审核`
- `已完成`

## 使用约束

- 默认使用中文词汇作为主文案。
- 日常页面不解释内部实现。
- 用户只需要理解 `AI 同事`、`模型连接`、`工作区`、`频道`、`任务` 和
  `审批`。
- `API Key` 只在设置里的连接表单出现。
- 旧的执行后端、兼容层、适配器名称只允许出现在内部架构文档或测试夹具中。
